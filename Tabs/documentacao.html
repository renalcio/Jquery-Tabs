Sobre

Autor Renalcio Carlos
Data 14/04/2014
Titulo JQuery Tabs
Versão 1.0
Linguagens HTML5 / CSS3 / JQUERY
Descrição Este plugin permite a criação de tabulações de conteudo de maneira fácil e prática, permitindo a utilização de HTML5 / XHTML para uso de tags exclusivas ou uso de tags padrões do HTML.

Primeiros Passos

Requisitos

Requisitos necessários para utilização do plugin:
  1. JQuery
  2. HTML
  3. CSS

Referências Externas

Inclua a referência a biblioteca JQuery
Ou

Referências Internas

Inclua a raferência ao plugin e ao tema CSS

Chamando o plugin

O chamado padrão do plugin é bem simples. 

$(fucntion)(){
    $("elemento").tab();
});

Estrutura HTML

O plugin pode ser estruturado de duas maneiras, XHTML (Tags Próprias) ou HTML (Tags padrão)

$(fucntion)(){
    $("tabs").tab({
        menu = "menu",
        abasConteudo = "aba",
        conteudo = "conteudo",
        principal = "tabs",
        itensMenu = "item",
    });
});

Adicionais

Posição Inicial

Você pode configurar a aba inicial da tabulação.

$(fucntion)(){
    $("tabs").tab({
        iniciante: 3 // Numero da aba que irá iniciar aberta.
    });
});

Parâmetros

Parâmetro Tipo Padrão Descrição Ver.
xhtml Bool true Habilita o uso de tags personalizadas na estrutura.1.0
iniciante Int 1 Aba inicial.1.0
menu String menu Tag que representa o menu de abas.1.0
itensMenu String item Tag que representa um item do menu (Será igual para os outros itens). 1.0
conteudo String conteudo Tag que representa o conjunto de conteudos. 1.0
abasConteudo String aba Tag que representa uma aba de conteudos (Será igual para os outras abas). 1.0
principal String tabs Tag que representa todo o conjunto a ser tabulado. 1.0


Estrutura completa


$(function(){
    $("elemento").tabs({
            iniciante: 1,           
            xhtml: true,            
            menu: "menu",           
            abasConteudo: "aba",           
            conteudo: "conteudo",   
            principal: "tabs",      
            itensMenu: "item",          
    });
});
